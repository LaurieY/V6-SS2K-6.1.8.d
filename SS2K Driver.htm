SkySensor 2000-PC ASCOM Driver Information.

Introduction.

Changes

Version 6.1.8a – 20 Aug 2013

Added an Option in Setup to force 1-Star Alignment on all syncs

Version 6.1.7c – 7 July 2011

No Functional changes. Altered install script using  Ascom V6 installer now no longer  needs to be in the ForcePlatformVersion list – Still install & works in Platform 5

 

Version 5.1.7c – 22 March 2010

Altered install  script to allow for comma separators & hence allow for Ascom versions with decimals in version number

Version 5.1.7b – 19 Dec 2008

Fixed many error return values and missing / incorrect properties to allow Conform V2.0.2 to run successfully. Also removed frame from version tab to fix windows theme problems

Allowed ROM < 2.10 to work with application based sync

Version 5.1.6d - 16 Nov 2008

Fixed required properties to return a value rather than Error in properties -  Get CanSetPierSide, CanSlewAltAz, CanSlewAltAzAsync, CanSyncAltAz

Version 5.1.6b – 13 July 2008

Fixed tracing to use the temp directory given in the system environment variable

Version 5.1.6 – 11 July 2008

Added the ability to be loaded via early binding mechanism implementing the iTelescope interface

This allows Metaguide 4.0.2 to work

Version 5.1.4 – 7 April 2008

Added a tab for Site Elevation to the setup tab with an allowed range of -300 to 10000 metres. 

 Value is saved in the registry and the SiteElevation property in Telescope.cls is implemented

Version 5.1.3 -  3 April 2008

Implemented Telescope.cls Property –Tracking and CanTrack.  Tracking implemented as a series of 1 sec timer events so probably only accurate within that timeframe.

Version 5.1.2 – 17 March 2008

Implemented Telescope.cls Property – IsPulseGuiding.  Implemented a pulse guide count to allow multi-axis guiding even though they are currently implemented as 1 axis at a time occurs..

Version 5.1.1 - 14 March 2008

Altered Telescope.cls Property – Sidereal Time to return value in hours, it had been returning as fraction of a day.

Version 5.1.0 – 13 March 2008

Altered Telescope.cls Property – Azimuth.  Previously the code subtracted 180 degrees from that returned by the mount.  This is not needed for Rom versions 2.05 up.  (and maybe earlier but not tested)

Version 5.0.0 – March 2008

Repackaged in platform 5 installer by Chris Rowland but essentially the same as that released with Ascom version4
